# DraftPhD
Need to install 
1. ATL: https://eclipse.dev/atl/
2. PRISM: https://github.com/prismmodelchecker/prism-api/tree/master
3. The Ensemble Language can be found in https://github.com/d3scomp/tcoef

Before to start use this project, please do some tutorial to gain more understanding.
1: You may do the simple ATL Transformation (This one requires ATL plugin and use direct Eclipse UI). https://wiki.eclipse.org/ATL/Tutorials_-_Create_a_simple_ATL_transformation
2: Once you understand the flow of step 1, you may try to run the ATL that's run programmatically. https://github.com/guana/ATLauncher/tree/master
